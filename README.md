# Mini Doc

Personal AI Assistant with document management and multi-platform messaging integration.

## Structure

```
├── frontend/          # Next.js frontend application
│   ├── src/          # Source code
│   ├── public/       # Static assets
│   └── ...           # Config files
└── README.md
```

## Quick Start

```bash
cd frontend
bun install
bun run dev
```

## Features

- AI-powered document assistant
- Multi-platform messaging (Telegram, WhatsApp, iMessage)
- Document upload and management
- App integrations (Gmail, Drive, Outlook, etc.)

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Bun
